# WhaleRadar - Smart Money & Accumulation Tracker

Aplikasi Streamlit tingkat lanjut untuk melacak aktivitas Whale dan Smart Money menggunakan API DexScreener dan analisis on-chain real-time.

## Cara Menjalankan Local
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan aplikasi: `streamlit run app/app.py`
